# Linux Commands

| Command | Description |
|----------|-------------|
| pwd | Show current directory |
| ls | List files |
| ls -l | Detailed list |
| ls -a | Show hidden files |
| clear | Clear terminal |
| whoami | Current user |
| hostname | Display hostname |
| date | Current date |
| cal | Calendar |
| history | Command history |
| man | Manual pages |
| echo | Print text |
| touch | Create file |
| mkdir | Create directory |
| rm | Remove file |
| cp | Copy |
| mv | Move/Rename |
